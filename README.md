# LoggerSoft
GD32E230C8T6 + BME280 + RA-01
# Подготовка среды
## IDE (Keil)
- Установить Keil5 с кряком (например, [отсюда](https://cloud.mail.ru/public/6Mi5/LJMCEYaYY))
## Пак и конфигуратор
- С [сайта](https://www.gd32mcu.com/en/download/0?kw=GD32E2) загрузить `GD32E23x AddOn` и `GD32 Embedded Builder`
-

| - | - |
| PA2 | UART0_TX |
| PA3 | UART0_RX |
| PA4 | RA_CS |
| PA5 | SPI0_SCK |
| PA6 | SPI0_MISO |
| PA7 | SPI0_MOSI |
| PA8 | BME_CS |
| PA9 | RA_RST |
| PA10 | RA_DIO0 |
| PA11 | RA_DIO1 |
| PA13 | SWDIO |
| PA14 | SWCLK |
| PB1 | ADC_BAT |
| PB12 | W25Q_CS |
| PB13 | SPI1_SCK |
| PB14 | SPI1_MISO |
| PB15 | SPI1_MOSI |
